Privacy Policy / プライバシーポリシー

Last updated: 2026-05-31

1. Application Overview / アプリケーション概要

threads-lead-finder is an internal research tool that uses the official Threads Graph API (https://graph.threads.net/) to identify Threads users who have publicly expressed interest in career consultation, job change, or related topics. The application is operated by the developer for internal lead research and outreach planning purposes. No automated messaging is performed by this application.

本アプリケーション「threads-lead-finder」は、Threads 公式 Graph API を使用して、 公開された Threads 投稿の中から、転職・キャリア相談に関心を持つユーザーを特定するための 社内リサーチツールです。本アプリケーション自体がメッセージを自動送信することはありません。

2. Data We Collect / 収集データ

The application collects only publicly available data returned by the official Threads Graph API:

  • Public post IDs, post URLs, and post text
  • Public usernames and profile URLs of the post authors
  • Post timestamps

No private data, direct messages, follower lists, email addresses, or personally identifying information beyond what is publicly visible on Threads is collected.

3. Data Use / データ利用目的

  • To identify potential leads for career consultation outreach.
  • To classify and score the leads using a deterministic keyword rule engine and/or the Anthropic Claude API.
  • For human review prior to any outreach. No automated DMs, replies, or contact of any kind are performed by the application.

4. Data Storage / データ保管

Collected data is stored locally on the operator's machine in a JSON database file (data/db.json) and exported CSV files. No data is uploaded to any third-party storage or server controlled by us.

5. Data Retention and Deletion / データ保持・削除

Collected records are retained only as long as needed for lead review (typically less than 30 days). Any individual may request deletion of their data by emailing hgf51805@gmail.com. Upon receiving a verified request, all corresponding entries will be deleted from local storage within 7 days.

6. Deauthorization and Data Deletion Requests / 連携解除・データ削除請求

If a user revokes the application's authorization on Threads, the application will cease using their authorization token. To request bulk deletion of collected public-post data related to your account, contact hgf51805@gmail.com.

7. Third Party Sharing / 第三者提供

Collected data is not sold, shared, or transmitted to any third party. For optional AI-based classification, the text of each public post is sent to the Anthropic Claude API solely to produce a categorical lead-rank label. This is the only third-party data flow, and no Threads user identifiers are sent.

8. Compliance / 法令順守

Use of this application is subject to the Threads Platform Terms, Meta Platform Policies, and applicable laws including Japan's Act on the Protection of Personal Information (APPI) and the EU GDPR where relevant.

9. Contact / お問い合わせ

For any inquiries about this policy or data handling, contact hgf51805@gmail.com.
